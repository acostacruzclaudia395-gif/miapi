# API Productos

## Descripción
API REST creada con Spring Boot para gestionar productos.

## Tecnologías
- Java 21
- Spring Boot
- JPA
- H2 Database
- Maven

## Endpoints

GET /productos → Listar productos  
POST /productos → Crear producto  
DELETE /productos/{id} → Eliminar producto  

## Ejecución

mvn spring-boot:run

## Base de datos

H2 Console:
http://localhost:8080/h2-console
